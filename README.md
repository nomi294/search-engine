# 🔍 Search Engine with Trie & Inverted Index

A console-based Search Engine developed in C++ using Data Structures and Algorithms (DSA). The project implements Trie Data Structure and Inverted Indexing to provide fast keyword searching and autocomplete suggestions from text documents.

---

## 🚀 Features

- 🔎 Keyword-based document searching
- ⚡ Fast autocomplete suggestions using Trie
- 📂 Document indexing from text files
- 📝 Search history management
- 🧹 Clear search history option
- 📑 Multiple document support
- 🎨 Colored console-based user interface

---

## 🛠 Technologies Used

- **Language:** C++
- **Data Structures:** Trie, Hash Maps, Sets, Vectors
- **Concepts:** Inverted Indexing, Searching, String Processing
- **Libraries:** STL, File Handling, Algorithms

---

## 📚 DSA Concepts Implemented

- Trie Data Structure
- Inverted Index
- Hashing (`unordered_map`)
- Set Intersection
- String Manipulation
- Sorting Algorithms
- Recursive Traversal
